Case Study 1: Utah DMV Site Redesign

The Problem

Utah DMV's current website has little to no correlation between on-screen imagery and theme to the services offered and the user navigation can be confusing.

User Feedback

"The site is really ugly"

"There are too many menu options, I'm not sure where to go?"

Navigation Redesign

We simplified the navigation into 3 main categories:

  1. Home
  2. Registration
  3. About Us

Usability Tests

Testing on our wireframes with our new navigation greatly increased the usability of the site.

Tasks that took the user 2+ minutes before they gave up on the current site, the user was able to complete in seconds on the redesign
